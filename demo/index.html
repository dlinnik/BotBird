BirdBot Demo /demo

Эта страница имитирует host-приложение. Кнопка «Чат с поддержкой» открывает встраиваемый виджет BirdBot через embed SDK. JWT берётся из DEMO_JWT в .env — как в проде, через токен Databird.

Как это работает:
1. Страница загружает /embed/v1.js с BirdBot-сервера
2. Токен приходит из /demo/env.js (переменная DEMO_JWT)
3. По клику открывает iframe /widget/ и передаёт token через postMessage
4. При смене страницы в меню вызывается updateContext() — в Telegram уйдёт актуальная link

Запуск: MongoDB + npm run dev → откройте /demo/
